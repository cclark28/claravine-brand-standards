# Claravine Brand Standards

**The Data Standards Company™ · Public Brand Resource Hub**
Version 1.0 · March 2025

---

## Overview

This repository hosts the Claravine public-facing brand standards guide, deployed as a static HTML site via GitHub Pages. It contains approved, non-sensitive brand assets and usage guidelines for color, typography, iconography, logo, and voice.

**Live site:** `https://[your-org].github.io/claravine-brand-standards/`

---

## Scope

This guide includes **only** approved public-facing materials:

- ✅ Color system (primary, secondary, gradient scales)
- ✅ Typography system (Poppins + Roboto)
- ✅ Iconography (Google Material Symbols Rounded)
- ✅ Logo usage rules and downloadable SVG assets
- ✅ Voice and tone guidance

**Excluded by policy:**
- ❌ Platform UI screenshots or product walkthroughs
- ❌ Internal tools, workflows, or processes
- ❌ Proprietary competitive materials
- ❌ Non-public photography or illustrations

## Deploying to GitHub Pages

1. Push this repository to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root `/`
4. Access at `https://[your-org].github.io/[repo-name]/`

No build step required — this is a fully static, single-file HTML site.
